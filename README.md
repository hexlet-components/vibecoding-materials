# vibecoding-materials

Материалы и наборы файлов для практики из курса по вайбкодингу.

## documents

Учебный набор для урока про Claude Cowork. Это папка `documents/` — пятнадцать
PDF, которые имитируют реальный бардак в папке небольшого ИП за квартал:
счета, чеки, договоры, банковские выписки, акт и пара посторонних файлов
(статья, инструкция к роутеру).

Особенности набора, на которых строится практика:

- имена файлов бессмысленные (`scan_001.pdf`, `IMG_20260228.pdf`,
  `Документ Microsoft Word.pdf`), а даты и суть документа лежат внутри PDF;
- есть два настоящих дубликата под разными именами
  (`scan_001.pdf` и `Документ_новый(1).pdf`; `чек.pdf` и `чек копия.pdf`);
- есть похожий, но не дубликат: счёт за тот же хостинг, но за другой месяц
  (`scan_004.pdf`) — проверка на то, что агент сверяет содержимое, а не имена.

### Как скачать

Нажмите зелёную кнопку **Code → Download ZIP** на странице репозитория и
распакуйте архив. Внутри будет папка `documents` — её и отдавайте агенту.

### Как пересобрать (необязательно)

Файлы сгенерированы скриптом `generate.py` (Python, библиотека `fpdf2`).
Запускать его не нужно: готовые PDF уже лежат в `documents/`.
